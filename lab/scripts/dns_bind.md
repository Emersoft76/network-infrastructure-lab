# 📡 dns_bind.sh – Configuração Básica do DNS Bind9

---

## 📝 Explicação técnica

Este script instala o Bind9 e cria uma zona DNS chamada `lab.local` apontando para `192.168.50.10`.

---

## 🧩 Requisitos

- Ubuntu Server 20.04+
- IP configurado: 192.168.50.10

---

## 🚀 Como executar

```bash
sudo chmod +x dns_bind.sh
sudo ./dns_bind.sh
```
